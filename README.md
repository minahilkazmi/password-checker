# Password Strength Checker
A Python tool that evaluates password strength based on cybersecurity best practices.
## What it checks
- Password length (8+ characters)
- Uppercase letters
- Numbers
- Special characters (!@#$%^&*)
## How to run
1. Install Python
2. Run: python password_checker.py
3. Enter a password when prompted
## Inspiration
Built after completing Harvard CS50 Cybersecurity course.
